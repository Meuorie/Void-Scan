# Void-Scan
Elite System Integrity & Forensic Auditor.

## Purpose
This tool performs a deep security audit of Linux systems to ensure file integrity and detect unauthorized changes.

## Usage
1. Open Terminal.
2. Clone: `git clone https://github.com/Meuorie/Void-Scan.git`
3. Navigate: `cd Void-Scan`
4. Run: `sudo python3 void_scan.py` (root recommended for full scan)
5. Quick one‑liner (download & run directly):
   ```bash
   wget https://raw.githubusercontent.com/Meuorie/Void-Scan/main/void_scan.py && sudo python3 void_scan.py
